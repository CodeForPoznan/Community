# Protokół z posiedzenia Zarządu Stowarzyszenia Code for Poznań w dniu 9 grudnia 2020 r.

9 grudnia 2020 r. w siedzibie Stowarzyszenia odbyło się posiedzenie Zarządu Stowarzyszenia Code for Poznań.

W posiedzeniu uczestniczyli następujący członkowie Zarządu:
* Jan Małecki
* Krzysztof Durczak
* Małgorzata Wojciechowska-Mieloch
* Paulina Tarnawska
* Tomasz Magulski

## Przebieg posiedzenia
* Prezes Stowarzyszenia otworzył posiedzenie rozpoczęto o godzinie 19:30 i rozpoczęto dyskusję.
* Postanowiono że przewodniczyć posiedzeniu będzie Jan Małecki, a protokołować będzie Krzysztof Durczak.
* Paulina Tarnawska zaproponowała włączenie do porządku obrad uchwały o możliwości zdalnych spotkań zarządu.
* Jan Małecki zaproponował włączenie do porządku obrad uchwały o wysokości opłaty członkowskiej i procedurze rozliczania członków Stowarzyszenia.
* Paulina Tarnawska podniosła temat poinformowania aktualnych wolontariuszy o możliwości przystąpienia do stowarzyszenia.
* Dyskusja na temat sprawdzania obecności podczas spotkań Stowarzyszenia.
* Rozmowa na temat promowania Stowarzenia wśród potencjalnych wolontariuszy.
* Przyjęto jednogłośnie uchwałę o zdalnym uczestnictwie w posiedzeniu zarządu Stowarzyszenia Code for Poznań.
* Przyjęto jednogłośnie uchwałę o składce członkowskiej Stowarzyszenia Code for Poznań.
* Posiedzenie zarządu zakończono o godzinie 20:50.
